# **App Name**: StudyGie AI

## Core Features:

- Intelligent Career Profiler Tool: Uses AI to reason through education, skills, and interests to suggest high-fit career roles and job matching scores.
- Secure Academic Identity: Full authentication suite for students to save their career discovery progress across sessions.
- Animated Learning Roadmap: AI-generated monthly progression plan visualizing steps to acquire missing professional competencies.
- Skills Gap Visualization: An interactive comparison dashboard highlighting current strengths versus industry-required skillsets.
- Historical Report Vault: Storage for previously generated career reports using a relational database for long-term tracking.
- Voice-First Career Counsel: Accessibility-focused feature allowing students to speak their interests and listen to their generated career roadmap.
- Glassmorphic Dash: A high-fidelity dashboard for managing multiple career path explorations simultaneously.

## Style Guidelines:

- A dark futuristic theme based on tech innovation. Primary Color: Electric Violet (#9E59FF) for high-tech vibrance. Background: Deep Space Navy (#0F0B17) providing high contrast for glass elements. Accent: Radiant Indigo (#595FFF) to create depth through analogous transitions.
- Font pairing: 'Space Grotesk' for tech-oriented headlines; 'Inter' for highly readable body text. This combines mechanical precision with modern usability.
- Thin-line neon icons with a subtle glow, focusing on tech, growth, and academic achievement motifs.
- A bento-style dashboard layout featuring glassmorphic panels and modular card units for different career insights.
- Fluid framer-style transitions, floating background particles, and pulsing neon accents to simulate active AI processing.
